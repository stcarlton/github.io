2022

Description

2022 is a 3D, arcade style top down shooter. Shoot zombies, level up, gain new abilities and increase your power to maintain pace with infinitely scaling enemies. Hunt down and kill elites to gain items that grant powerful abilities and makes each play through unique.

Technical Overview

2022 was developed in Unity game engine. Source code is entirely in C# and follows MVC architecture design pattern. Visual assets are entirely open source. A complete openGL build is playable in the browser by clicking on the "Load Game" button (performance depends on hardware).

Casino Simulation

Description

This casino simulation allows players to cash in and hit the casino floor, with options to play one of several casino activities. Players are immersed into a realistic casino, complete with 2D animations, poker chips and authentic casino rules.

Technical Overview

Casino Simulation is a WPF application using C# for the back end and XAML for the front. Source Code architecture follows MVVM design pattern, which is well documented using 10+ UML standard design diagrams.
