Rainman

Description

Rainman is an augmented reality tool that suggests both a betting strategy and a playing strategy based on the cards that have already been played. Rainman applies computer vision to recognize cards, utilizing the card positions to deduce whether each card is owned by the player, the dealer or another player. The popular card counting strategy "Hi-Lo" is employed to dynamically update betting and playing suggestions.

Technical Overview

Rainman is an iOS app developed with Unity game engine. Source code is C# and follows MVC design pattern. ARKit package is utilized for AR functionality.

2022

Description

2022 is a 3D, arcade style top down shooter. Kill zombies, level up, gain new abilities and try to keep up with infinitely scaling enemies. Hunt down and kill elites to gain items that grant random, powerful bonuses and makes each play through unique.

Technical Overview

2022 was developed with Unity game engine. Source code is entirely in C# and architecture follows MVC design pattern. All visual assets are open source. A complete, openGL build is playable in the browser by clicking on the "Play Game" button below (not available on mobile).

Casino Simulation

Description

This casino simulation allows players to cash in and hit the casino floor, with options to play one of several casino activities. Players are immersed into a realistic casino, complete with 2D animations, graphical poker chips and authentic casino rules.

Technical Overview

Casino Simulation is a WPF application that uses C# for the back end and XAML for the GUI. Source Code architecture follows MVVM design pattern, which is well documented with over 10 design diagrams that can be found in the repo.
